HttpClient:
GetExternalDataAsync, GetAllExternalDataAsync, GetOneExternalDataAsync:
Usa a biblioteca nativa do .NET para fazer requisições HTTP.
Métodos como GetAsync são usados para enviar requisições GET e EnsureSuccessStatusCode verifica se a resposta foi bem-sucedida.

RestSharp:
GetExternalDataWithRestSharpAsync, GetAllExternalDataWithRestSharpAsync, GetOneExternalDataWithRestSharpAsync:
Usa a biblioteca RestSharp para simplificar a comunicação com APIs RESTful.
Métodos como ExecuteAsync são usados para enviar requisições e obter a resposta como uma string.

Flurl.Http:
GetExternalDataWithFlurlAsync, GetAllExternalDataWithFlurlAsync, GetOneExternalDataWithFlurlAsync:
Usa a biblioteca Flurl.Http para fazer requisições HTTP de forma fluente e concisa.
Métodos como GetStringAsync são usados para enviar requisições e obter a resposta diretamente como uma string.
Com esses métodos, você tem três maneiras diferentes de fazer requisições HTTP na sua aplicação, cada uma utilizando uma biblioteca diferente, oferecendo flexibilidade e opções para diferentes cenários e preferências de desenvolvimento.
